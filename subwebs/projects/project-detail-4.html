Intelligent Library Book Delivery System

Abstract

This project is the minimal functional unit of a larger intelligent library concept. It includes standardized modular bookshelves, a book-retrieving robot with a lifting tray, and a mobile app for user commands. The system achieves automated book retrieval and delivery, enabling unmanaged library services in communities and schools. Users can request books via the app, and the delivery robot navigates to the designated shelf, retrieves the book with its sensor-activated tray, and delivers it to the user. This system is designed to improve accessibility for users with limited mobility and increase the efficiency of library management.

Key Components & Technology

  • Modular Bookshelves: Designed as movable units with a gear-and-rack mechanism for book ejection, controlled by an Arduino and activated by an ultrasonic sensor.
  • Delivery Robot: A line-following vehicle equipped with a lifting platform. The platform's height is adjusted by a 360-degree servo to interface with different shelf levels.
  • Mobile App: Developed using MIT App Inventor, the app allows users to search for books and send commands (book ID, seat number) to the system via Bluetooth.
  • Fabrication: Utilized a combination of 3D printing (for custom parts like gears) and laser cutting, with a robust frame built from aluminum profiles and wood.
